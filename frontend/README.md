# Cotizador Orion Pro

Proyecto Vite + React + Tailwind con export a PDF (html2canvas + jsPDF).


## Pasos para ejecutar


1. Extrae el zip.
2. En la carpeta del proyecto ejecuta:

   ```bash
   npm install
   npm run dev
   ```

3. Abre el navegador en la URL que indique Vite (por defecto http://localhost:5173)
